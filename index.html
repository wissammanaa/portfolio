Articles & Analyses
Cybersécurité · 2024

Montée en puissance des ransomwares ciblant les PME

En 2024, les attaques par ransomware ont continué d'augmenter, avec un ciblage accru des petites et moyennes entreprises, souvent moins protégées. Les groupes comme LockBit ou BlackCat ont perfectionné leurs techniques d'intrusion via phishing et exploitation de vulnérabilités VPN.

Source : CERT-FR / ANSSI
Cloud · 2024

Adoption du cloud hybride dans les organisations

Le modèle hybride (on-premise + cloud public) s'impose comme la norme en entreprise. Azure et AWS dominent le marché, tandis que les équipes IT doivent maîtriser à la fois les infrastructures locales et les services cloud pour assurer continuité et sécurité des données.

Source : Microsoft Azure Blog
Cybersécurité · 2025

Zero Trust : vers une sécurité sans périmètre

Le modèle Zero Trust ("ne jamais faire confiance, toujours vérifier") s'impose face à la multiplication des accès distants et des appareils mobiles. Il repose sur une authentification forte, une segmentation réseau et un accès au minimum nécessaire (principe du moindre privilège).

Source : ANSSI — Guide Zero Trust
IA & Automatisation · 2025

L'IA au service de l'administration système

Les outils d'IA s'intègrent progressivement aux plateformes d'administration (Microsoft Copilot for IT, GitHub Copilot) pour automatiser la détection d'anomalies, générer des scripts PowerShell ou accélérer le diagnostic d'incidents réseau.

Source : Microsoft Learn
Identité & Accès · 2024

MFA généralisé : l'authentification multifacteur devient standard

Face à l'explosion des vols de credentials, le MFA (Multi-Factor Authentication) est désormais recommandé par l'ANSSI pour tous les comptes à privilèges. Microsoft impose d'ailleurs le MFA sur Azure AD depuis 2024 pour les administrateurs.

Source : ANSSI
Réseaux · 2025

SD-WAN : gestion intelligente des réseaux d'entreprise

Le SD-WAN (Software-Defined Wide Area Network) permet d'optimiser dynamiquement le routage du trafic entre sites distants. Il offre plus de flexibilité que les liaisons MPLS traditionnelles, avec une visibilité et un contrôle centralisé via une interface logicielle.

Source : LeBigData.fr
